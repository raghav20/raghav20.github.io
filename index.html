About

I am a masters student at University of Waterloo. My Research focuses on making tools for Distributed Computing using frameworks such as Apache Hadoop, Apache Spark and applying various Machine Learning algorithms to detect outlandish behavior in connected systems. I'm interested in Machine Learning, Search/IR, NLP, Distributed Systems and writing scalable production code. Additionally I love playing with Data and extract patterns out of it. Before joining University of Waterloo as a masters student I worked with SAS Research and Development for around 2 years, there I worked as a software engineer in the Data Visualisation Research team. I have worked as an intern with companies like Lenovo, Honeywell and Havells. I write regularly about Distributed Systems and Machine Learning on my Blog as well as on social networking site Quora.

I love travelling and exploring new places. I love playing soccer and I am a die hard Zlatan Ibrahimovic fan. Soccer clubs I supoort are Manchestor United and paris saint germain.

Your Life is your message to the world, make sure it is inspiring.

Comparing Pregel Related Systems

Large Scale Graph Processing Framework GraphLab is an example of a Pregel-inspired system. It does not use the BSP model. Instead, it uses an asynchronous model suitable for computations with sparse data dependencies. GraphLab is written in C++ and was originally tailored for machine learning tasks. GraphLab provides an Update function which is analogous to the Map function in MapReduce. This function can read and update the sets of program state in a controlled fashion.
